== One Spark

One Spark is a collaboration tool, I developed as part of my Bachelor's Degree thesis. 


== Getting Started

1. Clone repository
       git clone https://github.com/sfroestl/onespark.git

2. Migrate Database and install gems
       $ cd /onespark_backend
       $ bundle install
       $ rake db:migrate

3. Go to http://localhost:3000/
